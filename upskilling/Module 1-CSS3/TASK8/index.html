visibility: hidden vs display: none

The two boxes below demonstrate the difference:

I am visible (normal flow)
This is hidden (visibility: hidden) — space preserved
This is removed (display: none) — no space
I am also visible — notice the gap above me

visibility: hidden → element is invisible but still occupies space in the layout.

display: none → element is completely removed from the document flow.
